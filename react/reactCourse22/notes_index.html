Props vs States
Props are 'properties' passed into a component in order for it to work correctly. The props received by components are 'immutable'.

States are 'values' managed by the component, similar to variables declared inside a function.
Any Time you are changing values that will be displayed/saved, you'll likely be using state. ex:

    function greeting(name) {
        const date = new Date()
        const hours = date.getHours()

        let timeOfDay
        if(hours >= 4 && hours < 12) {
            timeOfDay = "morning"
        } else if(hours >= 12 && < 17>) {
            timeOfDay = "afternoon"
        } else if(hours >= 17 && hours < 20) {
            timeOfDay = "evening"
        } else {
            timeOfDay = "night"
        }

        return `Good ${timeOfDay}, ${name}!`
    }

    console.log(greeting("Jeffy"))
  • State is a way for React to remember saved values from within a component.
  • Props are for passing data into a component
  • State is used to hold values (and 'remember' those values even when React re-renders a component)
  • Imutable items are unchangeable -- Props are immutable, State is mutable

What we get back when we useState() is an array with the first value saved as the state we are saving. A better way of doing it is by 'Destructuring the Array' that we can more easily retrieve the result.

I need to look into Array Destructuring in javascript.
the example below returns "Yes" (without the ', f()') -- it has been destructured 


    import React from "react"

    export default function App(
        // Destructuring of the Array ** a common standard is to name the function 'set' + the name of the state
            // so in this example the state is named 'isImportant' and the function is 'setIsImportant'
        const [isImportant, setIsImportant] = React.useState("Yes")
        return ( 
                <div className="state">
                    <h1 className="state-title">State is IMPORTANT<h1>
                    <div className="state-value">
                        <h1>{isImportant}</h1>
                    </div>
                </div>        
        )
    )
When working with state we cannot simply change set and change a variable, we use the built in 'func' ('f()' in the array) to change the state.
