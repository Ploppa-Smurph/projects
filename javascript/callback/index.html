Callback Tutorial



when we want to set paramaters in a callback function, ex:
setTimeout(action(secondAction), 5000) -- this would make the callback execute immediately
in order to execute a callback that has parameters then you must:
setTimeout(()=> action(secondAction), 5000) - by using the arrow function to state the callback is a function it no longer runs automatically

you can have multiple functions with parameters in your callback in this manner, you Must use the arrow functin '()=>' in order to begin each additional function
by using the arrow function I can use a function as a parameter in a callback
