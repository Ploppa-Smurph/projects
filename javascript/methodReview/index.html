  • Filter - get all items in a list that is <=; it returns 'true' or 'false' for each item
  • Map - takes the elements of an array and creates a new array with the elements that you mapped; ie: 'name' in the example
  • Find - allows you to find a single (1) object in an array, it will return the properties of that item, it will return the 1st 'true' item in the array it comes accross
  • ForEach - works similar to a 'for loop' and iterates over each element in an array
  • Some - will return 'true' or 'false' if ANY element in the array matches the criteria set in the function
  • Every - similar to 'some' but checks to see if every-single element is 'true'. Reduce method requires 2 parameters (in ex: script.js)
  • Reduce - performs an operation on the array and returns the output of the operation
  • Includes - checks whatever is in the '.includes()' method is in the array.
