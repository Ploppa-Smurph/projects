JavaScript Factory / Constructor Function
the JavaScript Factory makes a template to create objects. ex: 

    function BellBoy (name, age, hasWorkPermit, languages) {
        this.name = name;
        this.age = age;
        this.hasWorkPermit = hasWorkPermit;
        this.languages = languages;
    }
the 'constructor function' name MUST be capitalized ("BellBoy" in the ex: above)
the 'inputs/parameters' listed in the BellBoy constructor function match up with the property data contained in the object

to 'Initialize' or create an object we would: 

    let bellBoy1 = new BellBoy("Timmy", 19, true, ["French", "English", "Spanish"]);
this would create an object (bellBoy1) with the properties that are equal to the 'arguements' that we provided in the Initializer.
