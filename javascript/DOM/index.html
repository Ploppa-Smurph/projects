Ep02 DOM: Intro to DOM

The DOM catalogs a webpage into Individual Objects we can select and manipulate.
the DOM is a Tree Structure with object to select and manipulate the data contained.
Select elements in the DOM with document.firstElementChild;

Objects have both 'properties' and 'methods'
Properties describe the object.
Methods are the actions the object can perform. Methods use dot-notification (ex: .click())
When we run code to 'get' a value from an object that is a 'getter'. ex: car.color; to get the color of a 'car' object
We can also 'set' the value of an object ex: car.numberOfDoors = 0; -- this allows you to set the value and is a 'setter'

When we 'Call Method' ex: car.drive(); -- a method is a function associated with an object.
in ex01 we create a button -- we have tapped into the button's properties and .click() is a method associated with the button object
we can appendChild() and setAttribute()

    additional selectors
  1. .getElementById = manipulate an element by it's 'id' -- a single item
  2. .getElementBySelector = can select an element, a class, or an id
  3. .getElementsByTagName = plural - will manipulate all HTML elements with that tag ex: document.getElementsByTagName("li")"
  4. .getElementsByClassName = select the 'class' of elements (plural), it will find all elements and return as an array
  5. .querySelector = manipulate the 1st element by the element name, class, or id
  6. .querySelectorAll = returns an array of all items that satisfy that query
